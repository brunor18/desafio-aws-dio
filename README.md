## Introdução
Este relatório apresenta o processo de implementação de ferramentas na empresa Abstergo Industries , realizado por Bruno de Oliveira Ribeiro. O objetivo do projeto foi elencar 3 serviços AWS, com a finalidade de realizar diminuição de custos imediatos.

## Descrição do Projeto
O projeto de implementação de ferramentas foi dividido em 3 etapas, cada uma com seus objetivos específicos. A seguir, serão descritas as etapas do projeto:

Etapa 1: 
- Amazon RDS
- Banco de dados relacional em nuvem
- Será implementado um banco de dados relacional em nuvem para guardas as informações de cada remédio ao invés de um banco de dados relacional localmente, isso ajudará a reduzir os custos, reduzir problemas de manutenção e reduzir problemas de espaço

Etapa 2: 
- Amazon S3
- Serviço para armazenar objetos 
- Será implementado o S3 para guardar fotos de remédios, bulas e fotos de clientes cadastrados, essa tecnologia reduz lentidão no banco de dados e ajuda reduzir os custos, pois objetos que são colocados no serviço mas não são regularmente acesados, o serviço automaticamente muda para um plano (camada) mais barato

Etapa 3: 
- Amazon VPC
- Servico de rede privada
- Será implementado o VPC para adicionar uma camada de segurança nos servicos RDS e S3, além de reduzir custos, manutenções de segurança e conexão direta com os servicos RDS e S3, reduzindo bastante custos



## Conclusão
A implementação de ferramentas na empresa *Abstergo Industries tem como esperado seguranca, redução de custos, maior espaço físico e redução de manutenções*, o que aumentaria a eficiência e a produtividade da empresa. Recomenda-se a continuidade da utilização das ferramentas implementadas e a busca por novas tecnologias que possam melhorar ainda mais os processos da empresa.
